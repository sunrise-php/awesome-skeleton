# Awesome Skeleton for modern development on PHP 7.1+

## Installation (via composer)

```bash
composer create-project sunrise/awesome-skeleton app
```

## Run application

```bash
composer serve
```

[Open](https://github.com/sunrise-php/awesome-skeleton/wiki)
